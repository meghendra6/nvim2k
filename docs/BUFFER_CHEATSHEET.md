# 🗂️ 버퍼 관리 치트시트

## 빠른 참조 카드

```
┌─────────────────────────────────────────────────────┐
│            🔄 버퍼 내비게이션 (이동)                 │
├─────────────────────────────────────────────────────┤
│  Shift + H          →  이전 버퍼                     │
│  Shift + L          →  다음 버퍼                     │
│  <leader>ea         →  직전 파일로 전환 (:b#)        │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│            🔍 버퍼 검색 및 선택                       │
├─────────────────────────────────────────────────────┤
│  <leader>fb         →  버퍼 검색 (Snacks) ⭐         │
│  <leader>fr         →  최근 파일 목록                │
│  <leader>fs         →  프로젝트 텍스트 검색          │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│            ❌ 버퍼 닫기/삭제                          │
├─────────────────────────────────────────────────────┤
│  <leader>qb         →  버퍼 닫기 (:bw)               │
│  <leader>qd         →  스마트 삭제 (레이아웃 유지) ⭐ │
│  <leader>qo         →  다른 모든 버퍼 닫기 ⭐        │
│  <leader>fx         →  현재 버퍼 제외 모두 삭제      │
└─────────────────────────────────────────────────────┘

⭐ = 가장 자주 사용하는 단축키
```

## Snacks 버퍼 선택기 단축키

**`<leader>fb` 입력 후:**

```
┌─────────────────────────────────────────────┐
│  Enter         →  버퍼 열기                  │
│  Ctrl + x      →  수평 분할로 열기           │
│  Ctrl + v      →  수직 분할로 열기           │
│  Ctrl + q      →  Quickfix로 보내기          │
│  Esc           →  취소                       │
└─────────────────────────────────────────────┘
```

## 실전 시나리오

### 📌 시나리오 1: 여러 파일 작업 중 빠르게 전환
```
1. Shift + L / Shift + H로 순차 이동
2. 또는 <leader>fb → 파일명 입력 → Enter
```

### 📌 시나리오 2: 작업 정리 (버퍼 정리)
```
1. <leader>fb로 필요한 파일 확인
2. <leader>qo로 현재 버퍼만 남기고 모두 닫기
```

### 📌 시나리오 3: 두 파일 왔다갔다
```
1. 파일 A에서 작업
2. Shift + L로 파일 B로 이동
3. <leader>ea로 다시 파일 A로 복귀
```

### 📌 시나리오 4: 버퍼 하나만 안전하게 닫기
```
1. <leader>qd (스마트 삭제, 창 레이아웃 유지)
```

## 🎓 Pro Tips

1. **버퍼 번호로 직접 이동**: `:b3` 또는 `:b main.py`
2. **버퍼 목록 확인**: `:ls`
3. **수정된 버퍼 확인**: `:ls +` (+ 표시가 수정됨)
4. **숨겨진 버퍼 확인**: `:ls !` 

## 🔗 관련 단축키

### 창(Window) 관리
- `Ctrl + H/J/K/L`: 창 간 이동 (tmux 연동)
- `<leader>qs`: 현재 창 닫기
- `Ctrl + W, s`: 수평 분할
- `Ctrl + W, v`: 수직 분할

### 파일 탐색
- `<leader>ff`: 파일 찾기
- `<leader>fg`: Git 파일 찾기
- `<leader>fS`: 파일/버퍼 스마트 찾기
- `<leader>fl`: 현재 버퍼 라인 찾기
- `<leader>fr`: 최근 파일
- `<leader>ee`: 파일 탐색기 (mini.files)
- `<leader>eE`: 프로젝트 루트 탐색기
